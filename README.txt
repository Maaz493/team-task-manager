## Tech Stack
- Backend: Java Spring Boot, MongoDB, JWT
- Frontend: React.js
- Database: MongoDB Atlas
- Deployment: Railway

## Features
- JWT Authentication (Signup/Login)
- Role-based Access (Admin/Member)
- Project Management (Create, Delete)
- Team Management (Add/Remove Members)
- Task System (Todo/In Progress/Done)
- Dashboard with Stats
